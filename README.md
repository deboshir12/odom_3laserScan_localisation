# odom_3laserScan_localisation

Для запуска необходимо поместить пакет в рабочую дирректорию ROS2, а затем выполнить следующие команды:

```bash
colcon build --symlink-install
source install/setup.bash
ros2 launch robot_pkg display.launch.py
```

Далее неоходимо запустить симуляцию в Gazebo. Для того, чтобы заставить робота двигаться в газебо стоит плагин на телеоп

## Структура пакета

```text
robot_pkg/
├── config/          # rviz config, bridge config, nav2 config
├── launch/          # launch files
├── models/          # some test model files
├── robot_pkg/       # python nodes
├── worlds/          # worlds files
└──  urdf/           # robot model (xacro and autogenerated URDF format)
