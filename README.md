# odom_3laserScan_localisation

Для запуска необходимо поместить пакет в рабочую дирректорию ROS2, а затем выполнить следующие команды:

```bash
colcon build --symlink-install
source install/setup.bash
ros2 launch robot_pkg display.launch.py
```

Далее неоходимо запустить симуляцию в Gazebo. Для того, чтобы заставить робота двигаться в газебо стоит плагин на телеоп

## Структура пакета

```text
robot_pkg/
├── config/          # rviz config, bridge config, nav2 config
├── launch/          # launch files
├── models/          # some test model files
├── robot_pkg/       # python nodes
├── worlds/          # worlds files
└──  urdf/           # robot model (xacro and autogenerated URDF format)
```

## Команды для nav2

Построение карты 
```bash
roslaunch slam_toolbox online_async_launch.py use_sim_time:=true
```

Сохранение построенной карты
```bash
ros2 run nav2_map_server map_saver_cli -f ~/diplom/src/robot_pkg/maps/warehouse_map
```

После построения карты можно запускать nav2. Первая команда поднимает map_server. Вторая запускает коррекцию локализации (сейчас уже просто запускается из лаунча), третья остальную часть nav2. ЗАпускать в разных терминалах
```bash
ros2 launch robot_pkg map_server.launch.py map_yaml:='/home/deboshir/diplom/src/robot_pkg/maps/warehouse_map.yaml' 
```
```bash
ros2 run robot_pkg range_odom_corrector_map # пережиток эпохи 
```
```bash
ros2 ros2 launch nav2_bringup navigation_launch.py   params_file:='/home/deboshir/diplom/src/robot_pkg/config/nav2_params.yaml'    use_sim_time:=true
```
