# odom_3laserScan_localisation

Для запуска необходимо поместить пакет в рабочую дирректорию ROS2, а затем выполнить следующие команды:

```bash
colcon build --symlink-install
source install/setup.bash
ros2 launch robot_pkg display.launch.py
```

Далее неоходимо запустить симуляцию в Gazebo. Для того, чтобы заставить робота двигаться необходимо прописать в новом терминале команду команду
```bash
ros2 topic pub /diff_drive/cmd_vel geometry_msgs/msg/Twist "{linear: {x: 0.3}, angular: {z: 0.8}}"
```
пока только по кругу, в скором времени будет нормальная траектория, я доделываю
